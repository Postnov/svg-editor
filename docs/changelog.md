# SVG Editor Pro — Changelog

## [2.5.0] - 2026-01-28

### Изменено
- Цвет элемента применяется мгновенно при изменении в color picker
- Кнопка "Применить" заменена на "Сбросить" для возврата к оригинальному цвету
- Сохранение оригинальных цветов элемента для возможности сброса

---

## [2.4.0] - 2026-01-28

### Изменено
- Выделение элемента: контрастная розовая обводка через drop-shadow (не перекрывает цвет)
- Панель выбранного элемента теперь плавающая, появляется рядом с элементом
- Улучшен стиль плавающей панели: белый фон, тень, компактный размер

---

## [2.3.0] - 2026-01-28

### Изменено
- ViewBox: добавлены числовые поля X, Y, W, H в сайдбаре
- ViewBox: кнопка включения визуального редактирования
- ViewBox: компактные ручки (только углы, 8px)
- ViewBox: рамка точно вокруг SVG

---

## [2.2.0] - 2026-01-28

### Изменено
- Ограничен размер превью иконки (max 320x320px)
- ViewBox редактирование перенесено в сайдбар
- Рамка viewBox теперь позиционируется точно вокруг SVG
- Уменьшены размеры drag-ручек viewBox

---

## [2.1.0] - 2026-01-28

### Изменено
- Полный редизайн интерфейса: двухколоночный layout
- Сайдбар (340px) слева с инструментами
- Большое превью справа с адаптивным размером
- PNG размер теперь выбирается через dropdown у кнопки
- Визуальное редактирование ViewBox с drag-ручками (углы и стороны)
- Панель выбранного элемента с индивидуальным color picker
- Улучшенный UX при работе с элементами SVG

---

## [2.0.0] - 2026-01-28

### Добавлено
- Выбор размера PNG (64, 128, 256, 512, 1024, 2048, custom)
- Палитра цветов с 8 предустановленными цветами
- Добавление своих цветов в палитру
- Градиенты (линейный и радиальный)
- Настройка угла градиента (0-360°)
- Экспорт в Base64
- Экспорт в Data URI
- Модальное окно для кода с кнопкой копирования

---

## [1.5.0] - 2026-01-28

### Добавлено
- Кнопка "Копировать" для SVG кода
- Clipboard API с fallback для старых браузеров

## [1.4.0] - 2026-01-28

## [1.2.0] - 2026-01-28

### Добавлено
- Контроль толщины обводки (stroke-width)
- Слайдер от 0 до 10 с шагом 0.5
- Числовой ввод для точных значений до 20
- Синхронизация слайдера и поля ввода

## [1.1.0] - 2026-01-28

### Добавлено
- Поворот SVG на 90° влево и вправо
- Отражение по горизонтали и вертикали
- Индикация активного состояния для кнопок отражения

---

## [1.0.0] - 2026-01-28

### Добавлено
- Загрузка SVG файлов через drag & drop и выбор файла
- Цветовой пикер с hex-вводом
- Четыре режима перекраски: Fill, Stroke, Both, All-in-one
- Превью с шахматным фоном (transparency grid)
- Скачивание результата в SVG формате
- Скачивание результата в PNG формате (512×512)
- Кнопка сброса к оригиналу
- Адаптивный дизайн для мобильных устройств
- Современный UI с шрифтом Inter
- Градиентные кнопки и тени
