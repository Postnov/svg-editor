- [x] Загрузка SVG (drag & drop + выбор файла)

## План разработки (v2.0)

### Этап 1: Трансформации
- [x] **Поворот** — кнопки поворота на 90° влево/вправо
- [x] **Отражение** — горизонтальное и вертикальное отражение

### Этап 2: Обводка
- [x] **Толщина обводки** — слайдер для изменения stroke-width

### Этап 3: ViewBox
- [ ] **Редактирование viewBox** — изменение x, y, width, height для обрезки/масштабирования

### Этап 4: Работа с элементами
- [ ] **Выбор элемента** — клик по элементу SVG для выделения
- [ ] **Удаление элемента** — удаление выбранного элемента

### Этап 5: Экспорт
- [ ] **Копирование SVG** — копирование кода в буфер обмена
- [ ] **Выбор размера PNG** — выпадающий список размеров (64, 128, 256, 512, 1024, 2048, custom)
- [ ] **Base64** — экспорт в Base64 строку
- [ ] **Data URI** — экспорт в Data URI формат

### Этап 6: Расширенная работа с цветом
- [ ] **Палитра цветов** — сохранение часто используемых цветов
- [ ] **Индивидуальная перекраска** — клик на элемент → назначение своего цвета
- [ ] **Градиенты** — линейный и радиальный градиент вместо сплошного цвета
