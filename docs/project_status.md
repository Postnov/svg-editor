# SVG Editor Pro — Статус проекта

## Текущий статус
**Этап:** Разработка функционала
**Дата начала:** 2026-01-28

## Реализованный функционал (v1.0)
- [x] Загрузка SVG (drag & drop + выбор файла)
- [x] Выбор цвета (color picker + hex input)
- [x] Режимы перекраски (Fill, Stroke, Both, All-in-one)
- [x] Превью с шахматным фоном
- [x] Скачивание SVG
- [x] Скачивание PNG (512px)
- [x] Сброс к оригиналу
- [x] Адаптивный дизайн

## План разработки (v2.0)

### Этап 1: Трансформации
- [x] **Поворот** — кнопки поворота на 90° влево/вправо
- [x] **Отражение** — горизонтальное и вертикальное отражение

### Этап 2: Обводка
- [x] **Толщина обводки** — слайдер для изменения stroke-width

### Этап 3: ViewBox
- [x] **Редактирование viewBox** — изменение x, y, width, height для обрезки/масштабирования

### Этап 4: Работа с элементами
- [x] **Выбор элемента** — клик по элементу SVG для выделения
- [x] **Удаление элемента** — удаление выбранного элемента

### Этап 5: Экспорт
- [x] **Копирование SVG** — копирование кода в буфер обмена
- [x] **Выбор размера PNG** — выпадающий список размеров (64, 128, 256, 512, 1024, 2048, custom)
- [x] **Base64** — экспорт в Base64 строку
- [x] **Data URI** — экспорт в Data URI формат

### Этап 6: Расширенная работа с цветом
- [x] **Палитра цветов** — сохранение часто используемых цветов
- [x] **Индивидуальная перекраска** — клик на элемент → назначение своего цвета
- [x] **Градиенты** — линейный и радиальный градиент вместо сплошного цвета
